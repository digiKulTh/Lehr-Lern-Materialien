## Vorbereitung Arbeitsplatz und Beleuchtung

- [ ] Hintergrundsystem als Hohlkehle aufstellen

- [ ] Objektbeleuchtung 45° seitlich, geneigt, gleiche Leuchtstärke und Abstände zum Objekt

- [ ] Softboxen für weiches Licht und dezente Schatten

- [ ] Kamera mittig auf Stativ setzen, üblicherweise mit Neigung (z. B. 15°)

- [ ] Laptop für Remote-Steuerung aufstellen

- [ ] Kabelführung auf Stolperfreiheit prüfen

- [ ] Handschuhe & ggf. Reinigungsmittel (Pinsel) bereitlegen

- [ ] Hoch- oder Querformat einheitlich je Objekt

- [ ] Objekt zentriert und formatfüllend platzieren, frontal oder 45° nach links gedreht 

## Einstellung Kamera und Objektiv

- [ ] Kamera im manuellen Modus

- [ ] Speicherformat RAW

- [ ] mit Blitz Belichtungszeit >= 1/250 s (Blitzsynchronzeit)

- [ ] Blendenwert F8 bis F16

- [ ] Weißabgleich durchführen,

- [ ] bei Blitzeinsatz 5600 Kelvin oder ```Blitzsymbol``` wählen

- [ ] Manuellen Fokus wählen

- [ ] ISO 100 (Base-ISO) einstellen

- [ ] Bildformat 3:2

- [ ] Mindestanforderungen Kamera: Vollformat-Sensor, 24 Megapixel

- [ ] Mindestanforderungen Objektiv: Brennweite >= 50 mm, idealerweise ein Macro-Objektiv (z. B. 90 mm

## Vorbereitung des Fotografierens

- [ ] wesentlich für die Belichtung sind Helligkeit und Blende beim Blitzen sowie bei Dauerlicht zusatzlich die Belichtungszeit 

- [ ] bei Blitzlichteinsatz Belichtungszeit so wählen, dass ohne Blitz ein schwarzes Bild entsteht

- [ ] auf Über- bzw. Unterbelichtung prüfen (mithilfe des Histogramms)

- [ ] gleichmäßige Ausleuchtung des Hintergrundes prüfen

## Fotos anfertigen

- [ ] Alle relevanten Ansichten, Detailaufnahmen nach Bedarf

- [ ] Erstes Bild von einem Objekt mit Farbkarte aufnehmen
    
- [ ] Drehung um gedachte Achse im Uhrzeigersinn, in gleichen Winkeln, meist 45°

- [ ] Bei jedem Bild korrekte Positionierung des Objekts prüfen

- [ ] Bei jedem Bild manuell Fokus setzen/prüfen

- [ ] ‚Falsche' oder ungeeignete Bilder sofort löschen

- [ ] Bildurheberschaft und Lizenz in den Bild-Metadaten (IPTC) hinterlegen

- [ ] RAW-Bilder in TIFF-Bilder (unkomprimiert, Farbraum Adobe RGB(1998), 16 Bit) umwandeln

- [ ] Dateiname aus Objektinventarnummer und laufender Nummer bilden, z. B. ```003456#01```

## Preparing the workstation and lighting

- [ ] Set up the background as an infinity cove

- [ ] Object lighting 45° lateral, tilted, same luminosity and distances to the object

- [ ] Soft boxes for soft light and subtle shadows

- [ ] Place the camera at the center of the tripod, usually tilted (e.g., 15°)

- [ ] Set up laptop for remote control

- [ ] Check cable routing to prevent tripping

- [ ] Have gloves on hand and cleaning agent (brush), if necessary

- [ ] Portrait or landscape format depending on the object

- [ ] Place the object such that it is centered and fills the picture format, facing front or turned 45° to the left

## Camera and lens setting

- [ ] Camera in manual mode

- [ ] RAW storage format

- [ ] With flash exposure time >= 1/250 s (flash sync time)

- [ ] Aperture setting F8 to F16

- [ ] Perform white balance

- [ ] If using flash, select 5600 Kelvin or flash symbol

- [ ] Select manual focus

- [ ] Set ISO 100 (ISO base)

- [ ] Image format 3:2

- [ ] Minimum camera requirements: full-frame sensor, 24 megapixels

- [ ] Minimum lens requirements: focal length >= 50 mm, ideally a macro lens (e.g., 90 mm)

## Preparing for photography

- [ ] Brightness and aperture are essential for exposure during flash and for exposure time in the case of continuous light

- [ ] For flash lighting, select the exposure time such that a black image is created without flash

- [ ] Check for over or under exposure (using histogram)

- [ ] Check for uniform background illumination

## Taking photographs

- [ ] All relevant views, detailed images as needed

- [ ] Include a color card in the first photo of an object

- [ ] Rotate clockwise on the assumed axis, at the same angles, usually 45°

- [ ] Check correct positioning of the object for each image

- [ ] Manually set/check focus for each image

- [ ] Immediately delete incorrect or unsuitable images

- [ ] File image copyright and license in the image metadata (IPTC)

- [ ] Convert RAW images to TIFF images (uncompressed, Adobe RGB(1998) color space, 16 bits)

- [ ] Create file name from object catalog number and sequential number, e. g. 003456#01
